# Udemy-Clone
Udemy Clone Website

A responsive Udemy clone built using HTML and CSS, showcasing course listings, categories, and modern UI design.

This is a frontend clone of the Udemy homepage built using HTML and CSS. The project replicates the layout and design of Udemy, including the navigation bar, course listings, recommended topics, and footer section. It demonstrates my ability to recreate real-world website layouts and apply modern CSS layout techniques like Flexbox.

Features

Responsive navigation bar with logo, search bar, and icons

Course categories section

Featured banner section

Recommended courses section

Topics recommended for you section

Most popular courses section

Footer with multiple links

Clean and structured UI design

Technologies Used

HTML5

CSS3

Flexbox

Google Fonts

Font Awesome Icons

Purpose

This project was created to practice frontend development skills by recreating a real-world website layout. It helped improve my understanding of layout design, Flexbox, spacing, alignment, and UI structuring.
